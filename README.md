# PROYECTO FINAL: Carrito de Compras

Este proyecto es una aplicación creada para el curso "React.js" dictada por Sergie Code en Digital House.

Se utilizó Bootstrap y algunos estilos personalizados, también SweetAlert2 para popups, material ui para el ícono del carrito.

Para los productos se utilizó Fake Store API.

También se utilizó diferentes tipos de Hooks: (useState, useEffect, useReducer)

Para la navegación se utilizó React Router

También context y providers
